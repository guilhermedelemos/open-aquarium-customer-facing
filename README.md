# Open Aquarium: Customer Facing

This will be the main web application which the users will use. They can check all aquarium parameters and manage the account.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
